# Microsoft-Intune-Device-Management-Lab

This project demonstrates Microsoft Intune device management in a cloud-based environment. The lab focuses on enrolling Windows 11 devices, creating compliance policies, deploying configuration profiles, managing device groups, and deploying applications through Intune.

Objectives

Enroll a Windows 11 device into Microsoft Intune
Create and manage device groups
Configure compliance policies
Deploy configuration profiles
Deploy applications to managed devices
Monitor device status and policy compliance


Technologies Used
Microsoft Intune
Microsoft Entra ID
Windows 11
Microsoft 365 Admin Center
Endpoint Manager

Tasks Completed
1. Device Enrollment
Joined Windows 11 device to Entra ID
Enrolled device into Microsoft Intune
Verified successful device registration

2. Device Groups
Created a test device group
Added enrolled devices
Verified group membership


3. Compliance Policies

Configured:

Password requirement
Minimum OS version
Device compliance monitoring


4. Configuration Profiles

Configured:

Windows Update settings
Security settings
Device restrictions


5. Application Deployment

Deployed:

Google Chrome
7-Zip

Verified successful installation on managed device.


6. Device Monitoring

Reviewed:

Device compliance status
Device health information
Policy assignment status

Key Takeaways

This lab provided hands-on experience managing Windows devices through Microsoft Intune. It demonstrated how organizations use cloud-based endpoint management to enforce security policies, deploy applications, and maintain compliance across managed devices.
